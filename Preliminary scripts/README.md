# Bienvenido a SMEsDex

En esta versión preliminar, tendrás la oportunidad de experimentar con el código para la creación de los primeros indicadores del sector. Para comenzar, sigue estos pasos:

1. **Descarga los Notebooks**: Agrupa todos los Notebooks en una carpeta; este será tu entorno de trabajo.
2. **Ejecuta los Notebooks en el siguiente orden**:
   1. `Downloader.ipynb`
   2. `Data_Base_Creation.ipynb`
   3. `Indicators.ipynb`
   4. `Plots.ipynb`
3. **Explora la primera entrega de indicadores**: Analiza los resultados generados.

Estos indicadores se alinean estrictamente con el informe proporcionado por el DANE para el año 2023, utilizando la encuesta de microempresas EMICRON.

¡Disfruta de tu exploración en SMEsDex!
